## MEMBER FUNCTION POINTERS

A regular function pointer stores the address of a free function:

      void (*fp)() = &someFunction;
      fp();                          // call it

A MEMBER function pointer also stores the address of a function,
  but it belongs to a class, so it always needs an object to run on:

      void (Harl::*mp)() = &Harl::debug;
      (myObject.*mp)();              // call via object
      (ptr->*mp)();                  // call via pointer  <- we use this
